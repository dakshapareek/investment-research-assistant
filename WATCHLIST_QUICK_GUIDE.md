# Quick Guide: Editable Watchlist

## What's New

The "Keep an Eye On" box is now fully editable! Add and remove stocks as you like.

## How to Use

### Add a Stock
1. Click "Edit" button
2. Enter symbol (e.g., MSFT)
3. Enter name (optional, e.g., Microsoft)
4. Press Enter or click "Add"
5. Click "Done"

### Remove a Stock
1. Click "Edit" button
2. Click the red "×" on any stock
3. Click "Done"

### Analyze a Stock
- Just click any stock chip (when NOT in edit mode)

## Features

✓ Add unlimited stocks
✓ Remove any stock
✓ Saved automatically (localStorage)
✓ Persists across sessions
✓ Symbol auto-uppercase
✓ Duplicate detection

## Default Stocks

- AAPL (Apple)
- NVDA (NVIDIA)
- TSLA (Tesla)

You can remove these and add your own!
