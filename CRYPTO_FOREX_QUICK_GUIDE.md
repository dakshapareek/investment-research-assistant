# Quick Guide: Crypto, Forex & More

## Supported Formats

### Crypto
```
BTC-USD    Bitcoin
ETH-USD    Ethereum
SOL-USD    Solana
ADA-USD    Cardano
DOGE-USD   Dogecoin
```

### Forex
```
EURUSD=X   Euro/USD
GBPUSD=X   Pound/USD
USDJPY=X   USD/Yen
```

### Indices
```
^GSPC      S&P 500
^DJI       Dow Jones
^IXIC      NASDAQ
^VIX       Volatility Index
```

### ETFs
```
SPY        S&P 500 ETF
QQQ        NASDAQ-100 ETF
GLD        Gold ETF
```

## How to Use

1. Type symbol in search bar (e.g., `BTC-USD`)
2. Click eye icon to add to watchlist
3. Click chip to analyze

## Format Rules

- **Crypto**: Use hyphen (BTC-USD, not BTCUSD)
- **Forex**: Use =X suffix (EURUSD=X)
- **Indices**: Use ^ prefix (^GSPC)
- **Stocks/ETFs**: Just the symbol (AAPL, SPY)
